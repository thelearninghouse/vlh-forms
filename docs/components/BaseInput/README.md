
# BaseInput
The base component for all inputs

```html
<base-input id="text" v-model="submit.formField" label="Form Field Example"/>
```

## Props


- `label` ***String***

  Label for form item

- `id` ***String***

  ID attribute

- `v-model` ***String|number***

  What the form-field is binding to

- `validationName` ***String***

  

- `validation` ***String***

  

- `type` ***String***

  

- `optional` ***Boolean***

  

- `autocomplete` ***String***

  

- `role` ***String***

  

- `pattern` ***String***
