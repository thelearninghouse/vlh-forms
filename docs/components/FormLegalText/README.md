
# form-legal-text
Form legal text
It will dynamically load the school's name that is set in the admin of the TlhForms WP Plugin

```html
<form-legal-text/>
```

## Props


- `school` ***String***

  

- `textColor` ***String***

  

- `fontSize` ***String***

  

- `padding` ***String***

  

- `disableStyles` ***Boolean***
